video tag uses attributes like
       controls -- to get all control like play,pause,download etc..,
       muted -- to mute the audio by default
       autoplay -- to play the video on page load
       Google chrome will not support autoplay unless it is muted

 3. audio tag : This tag is used to display audios. it also uses source to display specific audio
        <audio controls>
            <source src="path">
        </audio>
4. track: track is used to assign captions or subtitles for videos, it uses 'label' attribute for name , 'src' attribute for source file, 'srclang' attribute to specify source language
